IKS FISE Persistence Store TDB Backend
==================
This bundle implements a jena tdb backend for persistence store that is defined through eu.iksproject.fise.stores.persistencestore.jena.IJenaPersistenceProvider.java interface.   


after 
>mvn >mvn clean install -DskipTests=true
you can upload the jars in target folder to felix
