Jersey front-end to the FISE engine
===================================

Goals for this sub-project:

- RESTful web services API to KReS for machines.

- Human-friendly HTML interface to quickly test the service and document the API.


Building from source
--------------------

  $ mvn -DskipTests install

Easy build and deploy
----------

After having started sling (FISE) go in the KReS root and type
  
  $ ./install.sh

then go to

  http://localhost:8080/kres
